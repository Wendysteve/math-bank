Test Page Home Button Generator

Automatically add home buttons to your test pages

How to Use:

  1. Enter your home page path (where the "Home" button should navigate to)
  2. Paste your test page HTML code in the left textarea
  3. Click "Add Home Buttons" to generate the modified version
  4. Copy the output code and save it as your test HTML file
Examples: "../index.html", "https://www.test4college.com/index.html", "https://www.test4college.com/adam/index.html"
Home buttons will be added in one location:
  • PDF controls (with other navigation buttons)
